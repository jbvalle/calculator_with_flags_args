# calculator_with_flags_arg

Input:
honey val1 operator1 val2 operator 2 val3 ... etc

Operators are:
+ (addition)
- (subtraction)
/ (division)
x (multiplication)
power (power)
root (root)

Features:
• Arguments in Terminal
• Endless number of values and operators

Bugs:
• No operator priotization
• Calculation only works from left to right
